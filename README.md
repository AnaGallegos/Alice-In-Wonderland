# Alice-in-wonderland

Let's create a histogram of the frequency of words in an extract from Alice in Wonderland.

Inside you'll see:
- List comprehension
- Dict comprehension
- Matplot

Which steps we'll take:

- Tokenize the text
- Remove punctuations symbols and suffixes
- Remove stop words
- Probability normalization
- Calculate the histogram
- Print the histogram
